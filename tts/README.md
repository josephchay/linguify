# Text-To-Speech Agent

---

## Overview

Linguify TTS (Text-to-Speech) model is a generative vocal agent designed specifically for different scenarios.

## Features

1. Conversational TTS: The agent can converse with the user from given text input in a natural sounding voice.
2. Emotional TTS: The agent can convey certain emotion aspects in the speech output.
   It should be able to at least take a minor pause (break) in between appropriate phrases.
